What is Anvil?

Anvil is a suite of Java classes that will help you in your GUI implementation. Anvil is swing based but designed to be easier to understand therefore with few lines of code you could create usefull windows for your application. You can view some examples here or download the code from GitHub and play with it.

Developing

Please, feel free to make your pull requests or give your ideas by sending an issue with the tag [IDEA] in the title.
