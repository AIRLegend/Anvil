Anvil

An opensource graphic library for Java

View on GitHub Download .zip Download .jar

Some examples:

Here you can see some examples of window creation.

Message window

This is the most simple class. It stands for creating alert messages. (One line of code per message)

Menu window

It stands for creating simple menu's and assigning actions to its buttons.

Data Input window

It stands for creating a window in which the user can write.

Window from image

A type of window that can be created directly by images (like .png) as well as its components (buttons)

Also, with this type of window and Menu window, an HTML panel can be added (to display websites, forms or a local .html file)

In addition to all of that, Check panels can be added to menus or windows for creating forms.
